# Howl

## Introduce

Easy for use process watchdog

## Usage

```shell
Easy for use process watchdog

Usage: howl [OPTIONS] <EXECUTE>

Arguments:
  <EXECUTE>

Options:
  -p <PATH>      [default: .]
  -h, --help     Print help
  -V, --version  Print version
```

```shell
howl -p "./src" "python tools/get_signal.py"
```
